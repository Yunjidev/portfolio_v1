## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/Yunjidev/portfolio_v1

# Access
$ cd portfolio_v1

# Run the project
$ pnpm run dev

# The server will initialize in the <http://localhost:3000>
```
